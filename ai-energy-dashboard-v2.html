Enerji Kazananı: Opus
Claude Opus en az enerji harcadı: 2.341×10⁻⁴ kWh. Ancak fark çok küçük — Codex ve Sonnet ile aradaki yüzde fark %3'ün altında. Tek başına anlamlı değil, binlerce test tekrarında fark ortaya çıkar.
📦
Kod Şişkinliği: Codex
GPT-5 Codex 68,923 byte HTML üretti — Sonnet'in 4.1× katı. 167 DOM node'a karşın Opus sadece 58 node kullandı. Büyük kod doğrudan yüksek enerji demek değil ama okunabilirlik ve bakım açısından ciddi fark.
🎨
Style Recalc Paradoksu
Opus 140, Sonnet 143 style recalc yaparken Codex sadece 23 yaptı. Bu Codex'in daha statik CSS kullandığını gösteriyor — ama enerji tüketimine neredeyse hiç yansımadı.
⏱️
Hız Kazananı: Codex
Toplam Puppeteer süresi açısından Codex lider: 3579 ms. Opus 57 ms daha geride. Bu fark kullanıcı deneyiminde hissedilemez — ancak Puppeteer metriklerinde Codex form doldurma hızında öne çıkıyor.
🧠
Bellek: Hepsi Eşit
3 model de 1.58–1.59 MB JS heap kullandı. Vanilla JS olduğu için bellek farkı oluşmadı — bu Puppeteer'ın sabit Chrome yükünden kaynaklanıyor, HTML karmaşıklığından değil.
